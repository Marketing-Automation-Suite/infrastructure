# ZoomInfo Configuration Guide

## Overview

ZoomInfo provides B2B contact and company database for lead generation (enterprise service).

## Prerequisites

- ZoomInfo enterprise account
- API access (contact sales)

## Step-by-Step Configuration

### Step 1: Get ZoomInfo Account

1. ZoomInfo requires an enterprise account
2. Contact [ZoomInfo Sales](https://www.zoominfo.com/contact)
3. Request API access
4. Complete enterprise onboarding

### Step 2: Configure API Access

1. Receive API credentials from ZoomInfo account manager
2. You'll receive:
   - Username
   - Password
   - Optional: API key (if available)

## Required Credentials

- **username**: Your ZoomInfo API username
- **password**: Your ZoomInfo API password
- **api_key**: Optional API key (if provided)

## Testing Connection

The system tests by calling the contact search endpoint.

## Capabilities

- Search contacts by criteria
- Search companies
- Enrich contact profiles
- Find decision makers
- Get company intelligence
- Access comprehensive B2B database

## Pricing

Enterprise pricing - contact ZoomInfo sales for details.

## Troubleshooting

- **401 Unauthorized**: Verify credentials
- **403 Forbidden**: Check account permissions
- **Rate Limits**: Enterprise accounts have higher limits
